## [0.1.0] - 2026-02-22

### Added
- Initial repository setup and organization
- Core documentation:
  - README with OpenHaven branding and vision
  - Introduction (docs/01-introduction.md) - What OpenHaven is and why it exists
  - Architecture (docs/03-architecture.md) - System design and deployment model
  - Vision document (plans/vision.md) - Long-term strategic vision
- Architecture Decision Records:
  - ADR-001: User-Owned Infrastructure model
  - ADR-002: Zero SaaS Dependencies
- Technical specifications:
  - Stack Decisions (plans/specs/stack-decisions.md) - Opinionated technology choices
- Roadmap with semantic versioning (plans/roadmap.md)
- Research archive (plans/research/initial-exploration.md)
- Updated CONTRIBUTING.md for documentation phase
- Directory structure for planning and specifications

### Changed
- Transformed generic GitHub template into OpenHaven-specific repository
- Updated all documentation to reflect OpenHaven's philosophy and goals
